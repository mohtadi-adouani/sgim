## Run Locally

Clone the project

```bash
  git clone https://gitlab.isima.fr/easow/sgim
```

Go to the project directory

```bash
  cd sgim
```

Install dependencies

```bash
  npm install
```

Start docker

```bash
  docker compose up -d
```

Start the server dev

```bash
  npm run start:dev
```



## Tech Stack

**Server:** Node, Express, TypeScript

**ORM:** [Sequelize](https://sequelize.org/)
